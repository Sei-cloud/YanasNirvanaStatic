What is a Russian manicure? ▾
A Russian manicure is a technique that includes the use of a specialized electric manicure tool to remove cuticle buildup carefully.
What is the difference between hard and builder gel? ▾
Builder gel can be a hard gel. Hard gel you can't soak off and have to file off whereas soft gel can be soaked off.
What is the difference between hard and regular gel? ▾

    Hard Gel

  • Thicker gel that protects nails and allows growth.
  • Needs to be refilled with each service.
  • Guaranteed to last 4-6 weeks.

    Regular Gel

  • A gel color that is painted over regular base.
  • Needs to be removed completely for each service.
  • Guaranteed to last 2-4 weeks.
